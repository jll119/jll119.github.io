Music Synthesisier - March 2022

Github Link

The music synthesisier is developed as part of the Year 3 Embedded Systems module at Imperial College London.

To make the music synthesiser work, the embedded software was written. Several real time tasks will need to be executed concurrently, such as detecting key presses and generating the output waveform. The keyboard is controlled using a ST NUCLEO-L432KC microcontroller module that contains a STM32L432KCU6U processor, which has an Arm Cortex-M4 core. Timing, dependency and shared data structures analysis are carried out to verify that our real-time system behaves properly even in the worst case conditions.

The core features of this synthesiser includes scanning the key matrix to find out which keys are pressed, using an interrupt to generate a sawtooth wave and using threads to allow the key scan and display tasks to be decoupled and executed concurrently.

Several advanced features were implemented, i.e. handshake (autodetection), polyphony, filters and real time tuner effect.
